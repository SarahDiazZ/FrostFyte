FrostFyte User Stories

1. Enhance Competitive Gaming

As user 1, eliminating gimmicks would be preferable so that the characters can be managed only on one platform.

Elaboration: Multiple platforms allow for more competitive gaming. As far as our project goes, additional platforms would be removed so that the characters only move along one flat surface. Eliminating platforms is highly desirable but not necessary.

Constraints: With only one platform, the game is less competitive with less objects to move around.

Effort Estimation: 2 person-hours

Acceptance test: The characters will be able to move only along one flat walk off stage. Meaning that there are no instances of characters coming off a stage.


2. Unique Character Design

As user 2, a more desirable game requires unique characters so that users can differentiate their character from their opponent’s character.

Elaboration: Changing the physical appearance of characters will make it easier for the player to distinguish their fighting character.

Constraints: none

Effort Estimation: 2 person-hours

Acceptance test: When playing the game, there will be a difference between opponents.


3. System Uniqueness

As user 3, a unique design is important so that there is a difference between the setting and characters of other games and for it to attract a user’s attention.

Elaboration: A lot of platform fighters games are very similar and the issue of game uniqueness arises.

Constraints: none

Effort Estimation: 4 person-hours

Acceptance test: The game will be visibly different from other platform fighter games including platform and character changes.
